### **Week 1 Achievements**

* Gained a clear understanding of **AWS Cloud** and its core service groups including Compute, Storage, Networking, and Database.

* Successfully **created and configured an AWS Free Tier account**, including multi-factor authentication (MFA) setup and IAM user management.

* Became familiar with the **AWS Management Console**, learned how to search for services, customize the dashboard, and manage resources efficiently.

* **Installed and configured AWS CLI**, and practiced basic commands to check account information and manage AWS resources.

* Created and managed **AWS Budgets**, learned how to monitor usage and costs, and cleaned up unused resources.

* Practiced **building a Virtual Private Cloud (VPC)** with Subnets, Route Tables, Internet Gateway, NAT Gateway, and Security Groups; deployed and connected EC2 instances.

* Explored and configured **VPN Site-to-Site connections**, understanding how to link on-premises networks with AWS VPCs.

* Improved **self-learning, troubleshooting, and cloud management skills** through hands-on AWS exercises.
